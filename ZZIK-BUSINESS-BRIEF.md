## 🔬 핵심 기술: GPS 무결성 알고리즘

### 문제: GPS 조작 방지
관광객이 실제로 병원/가맹점을 방문했는지 어떻게 확인?

### 해결: 5중 검증 시스템 (100점 만점)

```
1. Distance Check (40점)
   └─ Haversine 거리 계산
   └─ 50m 이내 = 만점
   └─ 예: 15m → 38점

2. Wi-Fi SSID (25점)
   └─ 병원/가맹점 등록된 네트워크 매칭
   └─ 일치 = 만점
   └─ 예: "GangnamClinic_5G" 일치 → 25점

3. Time Sync (15점)
   └─ 서버 시간과 ±5분 이내
   └─ 예: 정확한 시간 → 15점

4. GPS Accuracy (10점)
   └─ 위치 정확도 30m 이내
   └─ 예: 25m → 8점

5. Velocity Check (10점)
   └─ 보행 속도 (2.78 m/s) 이하
   └─ 차량/드론 조작 방지
   └─ 예: 0.5 m/s → 10점

────────────────────────────
총점: 96/100 ✅ PASS (60점 이상)
```

### 실제 시나리오

**✅ 정상 방문 (96점)**
```typescript
{
  distance: 15m,      // 38/40
  wifi: "일치",       // 25/25
  time: "정확",       // 15/15
  accuracy: 25m,     // 8/10
  velocity: 0.5m/s   // 10/10
}
→ 총 96점 → 체크인 승인
```

**❌ GPS 조작 시도 (32점)**
```typescript
{
  distance: 500m,     // 0/40 (너무 멀음)
  wifi: "불일치",     // 0/25
  time: "정확",       // 15/15
  accuracy: 50m,     // 0/10 (부정확)
  velocity: 15m/s    // 0/10 (차량 속도)
}
→ 총 15점 → 체크인 거부
```

## 📊 Unit Economics (고객당 경제성)

### 블렌디드 평균 (병원 + 프랜차이즈)

```
ARPU (월평균 매출)
└─ 950,000원

CAC (고객 획득 비용)
└─ 2,750,000원

LTV (36개월 생애 가치)
└─ 31,250,000원

LTV/CAC 비율
└─ 11.4 (우수! 3.0 이상이면 건강)

Payback Period (투자 회수 기간)
└─ 2.9개월

Gross Margin (매출총이익률)
└─ 91%

Churn Rate (월 이탈률)
└─ 5%
```

### 왜 좋은가?

✅ **LTV/CAC = 11.4**
- 3.0 이상 = 건강한 SaaS
- 11.4 = 매우 우수

✅ **Payback = 2.9개월**
- 6개월 이내 = 우수
- 2.9개월 = 빠른 회수

✅ **Gross Margin = 91%**
- 소프트웨어 비즈니스 표준: 70-80%
- 91% = 매우 높음 (변동비 적음)
