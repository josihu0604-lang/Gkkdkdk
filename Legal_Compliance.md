# ZZIK 법적 컴플라이언스 가이드

**버전:** 1.0  
**작성일:** 2025-11-11  
**목적:** 한국 법규 준수 전략 및 실행 계획

## 🎯 개요

ZZIK 플랫폼이 준수해야 할 한국 법규와 해결 전략을 정리한 문서입니다.

---

## 1️⃣ 특정금융정보법 (VASP 규제)

### 규제 내용
- 암호화폐 거래 시 가상자산사업자(VASP) 신고 필수
- 신고 없이 운영 시 5년 이하 징역 또는 5,000만원 이하 벌금

### ZZIK 해결책

**✅ 쿠폰 우선 정책**
```
관광객 리워드 체계:
1순위: 디지털 쿠폰 (할인, 무료 서비스)
2순위: 제휴사 포인트 (스타벅스, GS25 등)
3순위: 암호화폐 (선택적, 3rd party VASP 통함)
```

**구현 방법:**
- 플랫폼 내 암호화폐 지갑 없음
- 사용자가 원할 경우 Bithumb/Upbit API 연동 (VASP 등록 업체)
- ZZIK은 쿠폰 발행 플랫폼으로만 작동

**법률 검토:**
- 금융위원회: "쿠폰 발행은 VASP 대상 아님" (2024.03 가이드라인)
- 리스크 레벨: ✅ Low

## 2️⃣ 의료법 제27조의3 (환자 유인 등 금지)

### 규제 내용
- 의료기관 광고 시 사전 심의 필수
- 환자 유인 행위 금지 (5년 이하 징역 또는 5천만원 이하 벌금)
- 병원 간 비교, 보증성 광고 금지

**법률 근거**: [의료법 제27조의3](https://www.law.go.kr/LSW/precInfoP.do?precSeq=83537)

### ZZIK 해결책

**✅ 병원명 직접 노출 안 함**
```
앱 내 표시:
❌ "강남 세브란스 병원 - 성형외과 30% 할인"
✅ "강남역 인근 성형외과 - 외국인 친화 시설"
```

**구현 방법:**
1. 지도에 병원 아이콘만 표시 (이름 없음)
2. 클릭 시 "병원 정보" 페이지 (병원 자체 제공 콘텐츠)
3. 쿠폰은 병원이 직접 발행 (ZZIK은 플랫폼만 제공)
4. 광고 아님 명시: "환자 경험 개선 서비스"

**법률 검토:**
- 보건복지부: "플랫폼이 광고주가 아니면 예외" (2023.08 유권해석)
- ⚠️ **주의사항**: 법무법인 자문 시 "환자 유인 금지 해당 여부" 명시 질의 필요
- 리스크 레벨: ✅ Low

## 3️⃣ 전자금융거래법 (선불전자지급수단 규제)

### 규제 내용 (2024.9.15 개정 반영)
- 선불전자지급수단 발행 시 금융감독원 등록 필수
- 2024.9.15 개정으로 **'2개 업종 이상' 요건 삭제**
- 단, **등록 면제 요건** 존재:
  1. **가맹 1곳 한정 발행** (자사 가맹점에만 사용 가능한 쿠폰)
  2. **발행잔액 소액 기준** (구체적 금액은 시행령 확인 필요)
  3. **연간 총발행액 소액 기준**

**법률 근거**: [금융위원회 보도자료 (2024.9.15)](https://www.fsc.go.kr/no010101/82339)

### ZZIK 해결책

**✅ "스탬프/티어" 시스템 (금전적 가치 없음)**
```
기존 (금지):
- 1,000 포인트 = 1,000원
- 현금 전환 가능

ZZIK (허용):
- 10개 스탬프 = "실버 티어"
- 쿠폰 획득 자격 부여 (비금전적 혜택)
- 현금 전환 불가
```

**구현 방법:**
- 스탬프는 "게임 속 뱃지"와 동일 (게임산업법 예외)
- 티어 시스템: Bronze → Silver → Gold → Platinum
- 각 티어별 쿠폰 다운로드 권한 차등 부여

**ZZIK 적용 전략:**
- 각 프랜차이즈 본사가 **자사 가맹점에만** 사용 가능한 쿠폰 발행
- ZZIK은 쿠폰 "배포 플랫폼"이지 "발행인" 아님
- 분기별 쿠폰 총발행액 내부 모니터링 (임계값: 연 ₩500M 추정)

**법률 검토:**
- 금융감독원: "현금 전환 불가 시 전자금융업 대상 아님" (2022.12)
- ⚠️ **주의사항**: 법무법인 자문 시 "등록 면제 해당 여부" 정량 기준 확인 필요
- 리스크 레벨: ✅ Low

## 7️⃣ 경품 과세 (기타소득)

### 규제 내용
- 경품 제공 시 기타소득세 원천징수 의무
- 건별 기타소득금액 5만원 초과 시 과세
- 외국인 비거주자 원천세는 국적·조세조약에 따라 상이

### ZZIK 해결책

**✅ 쿠폰 권면가 상한 설정**

**과세 최저한 기준:**
```
기타소득 금액 = 수입 - 필요경비
경품의 경우 필요경비 0%가 일반적
따라서 권면가 ≤ ₩50,000이면 과세 제외
```

**ZZIK 쿠폰 설계:**
| Tier | 쿠폰 종류 | 권면가 | 과세 여부 |
|------|----------|--------|----------|
| Tier 1-2 | 커피·햄버거 | ₩5,000~8,000 | ❌ 비과세 |
| Tier 3 | 외식 쿠폰 | ₩30,000 | ❌ 비과세 |
| Tier 4 | 마트 상품권 | **₩49,000** | ❌ 비과세 (안전선) |
| Tier 5+ | 없음 | - | 세무 복잡도 회피 |

**원천징수 (₩50K 초과 시):**
- 소득세 20% + 지방세 2% = **22%**
- 외국인 비거주자: 국적·조세조약에 따라 상이 → **사전 테이블 작성 필요**

**법률 근거**: [기타소득의 과세최저한](https://taxly.kr/post/1387)

**구현 방법:**
```typescript
// src/utils/tax.util.ts

export const COUPON_TAX_CONFIG = {
  // 과세 최저한
  TAX_THRESHOLD: 50000,
  
  // 안전 마진 (₩50K 경계선 회피)
  SAFE_MAX_VALUE: 49000,
  
  // 원천징수율
  RESIDENT_TAX_RATE: 0.22,  // 20% + 지방세 2%
  
  // 외국인 비거주자 원천세 (조약별)
  NON_RESIDENT_TAX_RATES: {
    'CN': 0.20,  // 중국: 20%
    'JP': 0.20,  // 일본: 20%
    'US': 0.30,  // 미국: 30%
    'DEFAULT': 0.22
  }
};

export function calculateWithholdingTax(
  couponValue: number, 
  userNationality: string,
  isResident: boolean
): number {
  // 과세최저한 이하는 원천징수 없음
  if (couponValue <= COUPON_TAX_CONFIG.TAX_THRESHOLD) {
    return 0;
  }
  
  const taxRate = isResident 
    ? COUPON_TAX_CONFIG.RESIDENT_TAX_RATE
    : (COUPON_TAX_CONFIG.NON_RESIDENT_TAX_RATES[userNationality] 
       || COUPON_TAX_CONFIG.NON_RESIDENT_TAX_RATES.DEFAULT);
  
  return Math.round(couponValue * taxRate);
}
```

**액션 아이템:**
- [ ] 쿠폰 권면가 상한 ₩49,000 설정
- [ ] 외국인 원천세 테이블 작성 (중국·일본·미주·유럽)
- [ ] 세무사 자문 (연 1회 지급명세서 제출 요건 확인)
