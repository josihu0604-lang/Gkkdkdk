## 📌 Executive Summary

ZZIK은 **의료관광 × 프랜차이즈 자동화 플랫폼**으로, 중국/일본 관광객을 한국 병원 및 프랜차이즈 가맹점에 GPS 기반으로 자동 유입시키는 B2B SaaS 비즈니스입니다.

### 핵심 가치 제안

- **병원**: 외국인 환자 자동 유입, 쿠폰 발행으로 재방문 유도
- **프랜차이즈**: 관광객 대상 매출 증대, 실시간 방문 데이터
- **관광객**: 병원/가맹점 방문 시 스탬프 적립 → 특별 혜택

### 법적 컴플라이언스 (v7.0 업데이트)

| 법규 | 이슈 | 해결책 | 상태 |
|------|------|--------|------|
| **특정금융정보법** | VASP 규제 | 쿠폰 우선, 암호화폐는 3rd party VASP 통해서만 | ✅ 준수 |
| **의료법 27조** | 의료광고 금지 | 병원명 직접 노출 없음, 환자 경험 개선 초점 | ✅ 준수 |
| **전자금융거래법** | 포인트 규제 | "스탬프/티어" 시스템 사용 (금전적 가치 없음) | ✅ 준수 |
| **위치정보법** | LBS 사업자 등록 | 필수 등록 예정 | ⚠️ 진행중 |
| **표시광고법** | 광고 표시 의무 | "#광고" 공시 (3개 언어) | ✅ 준수 |

### GPS 무결성 알고리즘

**5가지 검증 요소 (100점 만점)**

1. **Distance** (40점): Haversine 거리 계산 (50m geofence)
2. **Wi-Fi SSID** (25점): 병원/가맹점 네트워크 매칭
3. **Time Sync** (15점): ±5분 이내 동기화
4. **GPS Accuracy** (10점): 30m 이내
5. **Velocity** (10점): 2.78 m/s (보행 속도) 이하

**통과 기준:** 60점 이상

```typescript
// 예시 점수 계산
const score = {
  distance: 38/40,  // 15m 거리
  wifi: 25/25,      // SSID 일치
  time: 15/15,      // 정확한 시간
  accuracy: 8/10,   // 25m 정확도
  velocity: 10/10   // 0.5 m/s
};
const total = 96; // PASS ✅
```

## 🚀 Go-to-Market 전략

### Phase 1: MVP 런칭 (Month 1-3)

- **타겟**: 강남/명동 병원 10개 + 프랜차이즈 20개
- **전략**: 3개월 무료 체험 + 1:1 온보딩
- **목표**: 첫 유료 전환 5개 고객

### Phase 2: Growth (Month 4-12)

- **타겟**: 서울 전역 확장 (병원 35개, 프랜차이즈 60개)
- **전략**: 케이스 스터디 마케팅, 의료관광 에이전시 파트너십
- **목표**: ARR 6.3억원

### Phase 3: Scale (Year 2-3)

- **타겟**: 전국 주요 도시 (부산, 대구, 제주)
- **전략**: 프랜차이즈 본사 직접 계약, 화이트라벨 솔루션
- **목표**: ARR 19.8억원
